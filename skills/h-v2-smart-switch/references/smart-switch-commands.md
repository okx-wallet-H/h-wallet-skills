# Commands Reference: H_v2 Smart Switch

This document details the CLI commands for the `h-v2-smart-switch` skill.

## 1. Assess Market Regime
```bash
h-wallet smart-switch assess --instId <symbol>
```
Analyzes current market volatility and trend to determine the best strategy.

## 2. Enable Auto-Switch
```bash
h-wallet smart-switch enable
```
Turns on background monitoring to automatically switch between Grid, DCA, and Sniper.
